# CRUNCHII
A command line tool to watch anime from crunchyroll

## 

### INSTALL

```bash
git clone https://github.com/lemonguy-104/crunchii.git
cd crunchii
sudo make install
```

### UNINSTALL

```bash
cd crunchii
sudo make uninstall
cd ../ && rm -rf crunchii
```

### UPDATE

```bash
cd crunchii
git pull
sudo make install
```

### DEPENDENCIES

mpv lynx grep awk youtube-dl fzf
